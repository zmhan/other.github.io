《人人都能用英语》一

发表于 2016-10-10   |   分类于 读书   |  

今天，开始读李笑来老师的《人人都能用英语》。其实很早就想读了，但耐不住自己的懒惰和沉不下心，或者更直接的说就是自己不努力上进吧。
很早就从笑来老师那里接触一些新的概念，每一次接触，总感觉自己豁然开朗，有醍醐灌顶之感。但遗憾的是，自己总是停留在接收上，从没有进一步的实践，左耳朵进，右耳朵出。因此这么多年，依旧没有什么进步。举个简单的例子，如今天读的这本书中说，要学会随时记录问题，只有这样，才会在之后的某一刻不至于错过答案。这个道理，高中时我就实践过，当时使用的是“错题本”，虽然很“笨拙”却非常有效。而这个道理和方法，李老师也在《把时间当作朋友》中认真讲过，我读过两遍，然而现在回过头来看，很遗憾，自己并没有真的读懂、学会，没有去践行。

2016的开年礼物

发表于 2016-01-13   |   分类于 随笔   |  

很久没有写东西了。一方面是因为自己在看些书，更主要的是自己在等待一件事。而等待本身，就算没有太多期待，也多少让人会感到不安。终于，在经历了无理由的推迟之后，我的等待变成了现实。虽然没有惊喜，但也让自己感受到了那种久违的快乐。
20天努力换来的成绩，足以让我认识到自己还没有失去进步的能力。经历了这么多年的昏昏噩噩，现在的自己有些太患得患失，总是信心不足。而这件事让我认识到，其实只要踏踏实实的认真去做了，我的学习能力依旧会迸发。这让我很开心，我想这是一个好的开始，它使我自我进化的步伐迈得更加坚定！
2016，我们走着瞧……

小站的成立

发表于 2015-12-07   |   分类于 随笔   |  

建站过程

这个网站是采用Hexo＋Github搭建的静态博客，其实愿是打算采用linode云主机＋Wordpress的。但在搜索学习的过程中，我发现自己所想要的只是一个能够安静的书写心情和记录成长的地方。于是，我改变了搜索方向，进而找到了这种可以满足我目前需求的静态博客搭建方式。

我是一个对科技类产品充满兴趣的人，但也是一个“好多都听过，什么都不懂”的小白。因此，在这个博客搭建过程中，网络起到了决定性的作用，感谢那些在网上分享知识的人，而建站的过程，也让我更加明白了“文档”的重要性。

既然打算把这儿当作记录我学习的地方，那么还是简单的记录下建站步骤吧，毕竟这是走出的第一步，也顺便熟悉下Markdown的语法。

  1. 安装git（Mac OS X下可以通过安装Xcode来完成）
  2. 安装node.js（见node.js）
  3. 安装Hexo（通过npm命令，详见Hexo）
  4. 安装NexT主题（通过git，配置方式见NexT）
  5. 申请Github账户并进行相关配置

期望和感谢

小站顺利建立了，这是一个好消息。希望这是一个好的开端，也希望自己可以坚持下去，以便实现自我的成长。

最后，谢谢各位；谢谢那些给我启发的人，不管你们知道与否、看到与否，由衷的谢谢你们；特别谢谢我最深爱的人，给我的大力支持；谢谢你们！
